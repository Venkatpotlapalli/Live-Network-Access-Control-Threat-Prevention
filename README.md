# Live Network Access Control & Threat Prevention Website

## 📌 Project Overview
This project is a Flask-based cybersecurity web application that controls network access, monitors login activity, detects brute-force attacks, and blocks suspicious IP addresses.  
It simulates a basic **Security Operations Center (SOC)** environment with user and admin dashboards.

---

## 🔐 Features
- User & Admin authentication
- Role-based access control
- Login activity monitoring
- Brute-force attack detection
- Automatic IP blocking
- Admin security dashboard
- Access logs for forensic analysis
- SQLite database integration

---

## 🛠 Technologies Used
- Python (Flask)
- HTML, CSS, Bootstrap
- SQLite
- Git & GitHub
- Linux (Ubuntu)
